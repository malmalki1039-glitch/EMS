# EMS System for Booking Classrooms

## Project Overview
This project is an Event Management System (EMS) designed specifically for booking classrooms within a Clinical Simulation Center. The application aims to streamline the process of reserving rooms for training and instructional purposes, making it easier for users to manage their bookings efficiently.

### Features
- **User Registration & Login**: Secure registration and authentication for users.
- **Classroom Availability**: Real-time display of classroom availability to prevent double bookings.
- **Booking Management**: Users can create, modify, and cancel bookings easily.
- **Admin Dashboard**: Special interface for administrators to manage classrooms and user bookings.
- **Notifications**: Alerts for upcoming bookings and reminders for cancellations or modifications.

### Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB
- Deployment: Heroku

### Getting Started
To get started with this project, clone the repository and install the necessary dependencies:
```
git clone https://github.com/malmalki1039-glitch/EMS.git
cd EMS
npm install
```
